# Excel Sales Dashboard

## Project Overview
This project presents an interactive sales dashboard created using Microsoft Excel to analyze sales performance and trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Cleaning

## Key Features
- Sales analysis by region and product category
- Monthly sales trend visualization
- Interactive filters using slicers
- KPI metrics for total sales and profit
